## Featured Project

### 🔬 Identification of Optical Counterparts in Globular Clusters

A full pipeline that combines **optical CMD analysis** and **X-ray data** to identify probable counterparts of X-ray sources in globular clusters.

<p align="center">
  <img src="https://raw.githubusercontent.com/AngelMP06/-Identification-of-Optical-Counterparts-in-Globular-Clusters/main/results/NGC_6809/candidates/CMD_275_336_counterparts.png" width="45%">
</p>

🔍 **What this project does:**
- Classifies stellar populations using Color–Magnitude Diagrams (CMDs)  
- Analyzes X-ray sources and their properties  
- Cross-matches optical and X-ray catalogs  
- Computes probabilities for candidate counterparts  

👉 [View Project](https://github.com/AngelMP06/-Identification-of-Optical-Counterparts-in-Globular-Clusters)
